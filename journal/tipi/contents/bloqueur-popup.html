Désactiver le bloqueur de fenêtre publicitaire (pop-up)

Débloquer les popups sur les navigateurs Internet Explorer:

Comme indiqué sur l'image ci-dessous, cliquez sur "Outils" puis sur "Bloqueur de fenêtre publicitaires " et cochez "Désactiver le bloqueur de fenêtres publicitaires"

Débloquer les popups sur les navigateurs Firefox:

Comme indiqué sur l'image ci-dessous, cliquez sur l'icône "Menu" puis sur "Options"

 

Ensuite, l'onglet présent ci-dessous s'ouvre. Séléctionnez l'élément "Contenu" puis décochez la ligne nommée "Bloquer les fenêtres popup". Vous pouvez fermer l'onglet.

Débloquer les popups sur les navigateurs Chrome:

Comme indiqué sur l'image ci-dessous, cliquez sur l'icone "Menu" puis "Paramètres"

 

Ensuite, la page présente ci-dessous s'ouvre. Descendez vers le bas, puis cliquez sur le lien "Afficher les paramètres avancés..."

 

Dans la partie déroulée, en-dessous du titre "Confidentialité", cliquez sur le bouton "Paramètres de contenu..."

 

Dans la fenêtre qui s'ouvre, descendez jusqu'au titre "Fenêtres pop-up". Sélectionnez "Autoriser tous les sites à afficher des fenêtres pop-up". Cliquez sur le bouton "OK" pour enregistrer la modification.
NB. : Vous pouvez à tout moment refaire la manipulation et cocher "Interdire à tous les sites d'afficher des fenêtres pop-up" si vous souhaitez ne plus afficher de popups.
